Summary:c:geo - calendar (add-on)
Description:
This is a calendar plugin for the c:geo app.
It will only work with the c:geo app installed and allows to add event caches to your personal calendar.

Functionality:
When you open an event cache, press menu and add to calendar. A new appointment for the event date (and time) will be created in your calendar.

Remark:
This plugin does not work on some Samsung devices on Android 4.0+ due to a Samsung specific problem in the calendar.
